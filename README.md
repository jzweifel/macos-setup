### Screenshots

- `$ mkdir -p ~/screenshots`
- `$ defaults write com.apple.screencapture location ~/screenshots && killall SystemUIServer`

### Show Hidden Files

- Type this into your terminal: `defaults write com.apple.finder AppleShowAllFiles YES`
